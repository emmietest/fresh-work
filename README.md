# Fresh Work

A React + Vite + Tailwind CSS project.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This project is configured for automatic deployment to GitHub Pages.

1. Push your code to the `main` branch of your GitHub repository.
2. Go to your repository settings > Pages.
3. Set the source to "Deploy from a branch" and select `gh-pages` branch.
4. The GitHub Actions workflow will automatically build and deploy your site.

For user/organization pages (username.github.io), ensure your repository is named `username.github.io`.
